Welcome to the Team 1 Data Import Utility

This demonstration will be going over the data import utility. We will be loading an empty database with new and useful data that we can then view using the supplemental API. When the data import utility would be integrated fully into the Hotel Scheduling System, only the Data import utility page would be integrated somewhere into the frontend team's website and used accordingly. The API in general will find itself under the hood in many other places as well.
